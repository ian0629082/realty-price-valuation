## 六、計算公式

請依照以下公式計算。

### 1. 總容積

```text
總容積 = 土地坪數 × 容積率(%) × 免計容積倍率
```

因為容積率在程式資料中可能是 `300`，代表 `300%`，所以計算時要轉換成：

```text
容積率倍率 = 既有容積率欄位 / 100
```

實際公式為：

```text
總容積 = 土地坪數 × (既有容積率欄位 / 100) × 1.3
```

---

### 2. 土地倍數

```text
土地倍數 = 總容積 / 土地坪數
```

---

### 3. 土地單坪價

```text
土地單坪價 = 土地買價 / 土地倍數
```

土地買價的單位是：

```text
萬 / 坪
```

---

### 4. 營造成本

```text
營造成本 = 建物興建成本 + 土地單坪價
```

目前建物興建成本固定抓：

```text
20 萬 / 坪
```

所以：

```text
營造成本 = 20 + 土地單坪價
```

---

### 5. 預計售價單坪價

```text
預計售價單坪價 = 營造成本 × 管銷倍率 × 利潤倍率
```

也就是：

```text
預計售價單坪價 = 營造成本 × 1.15 × 1.2
```

## 八、範例測試

假設使用者輸入：

```text

土地買價 = 120 萬 / 坪

```

其中：

```text

```

```text

```

```text

土地倍數 = 3.9
```

```text

```

```text

```

```text
預計售價單坪價 = 50.77 × 1.15 × 1.2
預計售價單坪價 = 70.06 萬 / 坪
```

因此畫面應顯示：

```text
總容積：390.00 坪

土地單坪價：30.77 萬 / 坪
```
